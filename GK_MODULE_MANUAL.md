# 中国古代名句辞典与背诵模块用户手册

## 1. 简介
本模块是自我学习系统中的“通识模块”，旨在帮助用户学习、积累和背诵中国古代名句。通过科学的间隔重复算法（SRS），帮助用户高效记忆。

## 2. 功能特点
*   **名句库管理**：浏览、搜索已收录的名句。
*   **智能背诵**：基于记忆曲线的背诵复习系统。
*   **PDF导入**：支持从PDF文档批量导入名句。
*   **学习统计**：可视化展示学习进度和掌握情况。
*   **多端同步**：支持数据在本地和服务器之间同步。

## 3. 使用指南

### 3.1 进入模块
1.  在系统左侧导航栏点击“学习”。
2.  在学习系统主页找到“通识模块”卡片。
3.  点击卡片进入模块主页。

### 3.2 浏览名句
*   默认显示“名句列表”标签页。
*   顶部搜索框可输入关键词（内容或作者）进行筛选。
*   列表展示名句原文、作者、出处及释义。

### 3.3 导入名句
1.  点击右上角的“导入PDF”按钮。
2.  在弹窗中点击“选择文件”，上传包含名句的PDF文件。
3.  点击“解析文本”，系统将自动提取文本并在下方预览。
4.  确认预览内容无误后，点击“确认导入”。
    *   *提示：系统会自动识别格式，建议PDF内容格式清晰，每行一条名句。*

### 3.4 开始背诵
1.  点击“背诵挑战”标签页。
2.  系统会自动筛选今日需要学习的新词和复习的旧词。
3.  **正面**：显示名句原文。
4.  **点击卡片**：翻转到背面，显示作者、出处和释义。
5.  **自我评分**：
    *   **忘记**：重置进度，尽快复习。
    *   **模糊**：缩短复习间隔。
    *   **记得**：按计划延长复习间隔。
6.  当今日任务完成后，显示完成提示。

### 3.5 设置计划
1.  点击右上角的“设置”按钮（齿轮图标）。
2.  **每日新词上限**：设置每天想要学习的新名句数量（默认10）。
3.  **每日复习上限**：设置每天复习的最大数量（默认50）。
4.  点击“保存设置”生效。

### 3.6 查看统计
1.  点击“学习数据”标签页。
2.  查看总收录数、熟记数（连续正确5次以上）。

## 4. 常见问题
*   **Q: 导入PDF乱码怎么办？**
    *   A: PDF格式复杂可能导致解析不准，建议手动复制文本到预览框进行调整后再导入。
*   **Q: 数据丢失了？**
    *   A: 数据存储在浏览器本地，同时也尝试同步到服务器。请确保不要随意清除浏览器缓存。
